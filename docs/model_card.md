# Model card

## 模型概述

- 骨干：DINOv2 ViT-S/14
- 分类头：融合 patch token 全局平均池化与最大池化的 SE Head
- 训练策略：先冻结骨干训练分类头，再解冻最后两个 Transformer block 和归一化层微调
- 二分类正类：Hazard

## 预期用途

用于消防隐患图片的比赛评测、研究验证和辅助筛查。模型输出不能替代专业消防检查，也不应用作无人监督的安全处置依据。

## 局限性

- 测试集仅 44 张图片，统计置信度有限。
- 多分类测试集与验证集规模较小，100% 准确率不代表真实场景中的泛化性能。
- 光照、遮挡、摄像头域差异和未见类别可能导致性能下降。
- 数据集不允许公开，公共仓库用户无法独立获得完全相同的测试图片。
